# RETRACE
Retrace is a mobile application developed by a collaborative software engineering team at Makerere University. The app is designed to help users track, report, and recover lost and found items within a community or campus setting.

## Project Overview
The project, specifically the Retrace application, focuses on solving the problem of lost property by providing a centralized digital platform. It integrates modern mobile development practices with cloud-based backend services to ensure data persistence and user security.

### Key Features
User Authentication: Secure Login and Sign-Up flows powered by Firebase Authentication.
Item Reporting: A dedicated interface for users to submit details about lost or found items.
Cloud Integration: Real-time data storage and management using Firebase Firestore.
Modern UI/UX: A responsive and intuitive interface built using Jetpack Compose and Material3 design principles.
Branded Experience: Custom splash screen and consistent brand coloring (Retrace Navy and Mid-Blue).

## Technical Stack
Language: Kotlin.
UI Framework: Jetpack Compose.
Backend: Firebase (Auth and Firestore).
Dependency Management: Gradle (Kotlin DSL/Groovy).
Version Control: Git (GitHub).

## Setup and Installation
Clone the Repository: Use Git to clone the project from the team's GitHub repository.
Firebase Configuration: Add the necessary google-services.json file to the app directory to enable backend services.
Gradle Sync: Open the project in Android Studio and synchronize the Gradle files to download dependencies.
Deployment: Connect an Android device via USB or Wireless Debugging to run the application.
